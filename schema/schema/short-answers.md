1. Difference between Google Knowledge Graph and Google Knowledge Panel

The Google Knowledge Graph is a large database of connected entities and relationships between people, places, and things. It is the backend system that stores structured information.

The Google Knowledge Panel is the visible result shown on Google search pages when a specific entity is searched. It is powered by the Knowledge Graph.

2. How Google determines entity identity

Google determines entity identity using:

Consistent name usage across the web
Structured data (Schema markup)
Backlinks and mentions from trusted websites
Social profile consistency (sameAs links)
Content relevance and contextual signals
3. When to create Custom Post Types instead of pages

Custom Post Types are used when managing structured, repeatable content such as:
Services
Portfolio items
Blog posts
Products

Pages are used for static content like About, Contact, and Home pages.

CPTs are better for scalable and dynamic content structures.

4. Recommended plugins for speed optimization and why
WP Rocket: Improves caching, reduces load time, and optimizes files
LiteSpeed Cache: Server-level caching for faster performance
Autoptimize: Minifies CSS, JS, and HTML for faster loading
Smush: Compresses images without quality loss
These plugins improve Core Web Vitals and overall site performance.
